## NEWSLETTER SIGN-UP FORM 📰

A responsive, dark theme newsletter signup component built with *html* and *css*. It features an email input field, a styled subscribe button, and a list of benefits.

---------------------

##  Features 🚀

- Dark mode design with modern color palette.
- Email input field with icon.
- Responsive layout(mobile-friendly).
- Hover animation on the subscribe button.
- Benefits section with check icons.

-------------------------

## Technologies 🛠️ 

1. **HTML**
2. **CSS**  (Flexbox, media query)
